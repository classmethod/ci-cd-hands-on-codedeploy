ハンズオンで構築する構成

ハンズオンで構築する構成

今回は上記の図のような構成を構築します。

  • GitHub にコードがプッシュされると CodePipeline での処理が開始されます。
  • テスト実行用 CodeBuild ではテストを実行します。
  • アラートチェック用 CodeBuild ではアラートが発報されていないかチェックします
  • CodeBuild での処理が成功したら EC2 に新しいバージョンのイメージがデプロイされます。
